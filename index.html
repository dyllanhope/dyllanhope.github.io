Front End

Creating a matching-dates page

The task was to create an app that has 2 date inputs and a row of blocks, 1 for each weekday. The user enters the first date(eg. 01-01-2018) into the first input, the app then highlights which day of the week that date would fall on (eg. Monday) in blue. The same would occur for the second date input, but would just highlight the weekday in red rather. If both date inputs happened to fall on the same weekday, then that weekday is highlighted in green. All changes to classes (colours) and element creation is done using handlebars templates

link: https://dyllanhope.github.io/matching-dates/

Creating a Registration page

This is a web page that takes in a users registration an keeps the data in a list that is then displayed on screen in a similar design to a number plate. The user is then able to filter the saved registrations according to towns. The web page then loads in the registrations from the selected towns. The registrations also remain through a page refresh. The page is also split into 2 halves; the left half was built using regular JavaScript, HTML and CSS and the right half was built using handlebars templates.

link: https://dyllanhope.github.io/Registration/

Creating a greetings page

The link below will send you to a page that will expect 2 types of input; one is expecting you to enter a name, the other is a set of radio buttons asking for your language.
